## Test Tools

- [Jest](https://jestjs.io/)
- [Cypress](https://www.cypress.io/)
- [wallaby.js](https://wallabyjs.com/)

## Commands

### Local Development
```bash
$ npm run start
```

### Build the Project
```bash
$ npm run build
```

### Test the Project
- Unit Test
```bash
$ npm run test:unit
```

- Unit Test Watch
```bash
$ npm run test:unit:watch
```

- Unit Test Watch All Files
```bash
$ npm run test:unit:watchAll
```

- Test Coverage
```bash
$ npm run test:unit:coverage
```

- Unit Test UI
```bash
$ npm run test:unit:ui
```

### Lint your Project
- Lint the src folder
```bash
$ npm run lint:src
```
- Lint the test folder
```bash
$ npm run lint:test
```
